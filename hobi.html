Hobi Saya

Selain berfokus pada perkuliahan di bidang Teknik Informatika, saya juga memiliki beberapa hobi yang membantu saya untuk tetap kreatif, produktif, dan terhibur dalam keseharian. Hobi bagi saya bukan hanya sekadar kegiatan mengisi waktu luang, tetapi juga cara untuk mengembangkan diri.

1. Bermain Game

Saya senang bermain game, terutama game strategi dan petualangan. Bermain game bukan hanya hiburan, tetapi juga melatih kemampuan berpikir cepat, kerja sama tim, serta strategi dalam menyelesaikan masalah.

2. Coding / Ngoding

Sebagai mahasiswa Teknik Informatika, saya suka menghabiskan waktu mencoba membuat program sederhana. Mulai dari membuat website dengan HTML & CSS, hingga mempelajari bahasa pemrograman Python. Bagi saya, ngoding itu seperti bermain puzzle, semakin sering mencoba semakin asik.

3. Membaca Buku & Artikel Teknologi

Saya juga hobi membaca buku maupun artikel online yang berkaitan dengan teknologi, terutama tentang pemrograman, perkembangan gadget, dan inovasi digital terbaru. Hal ini membantu saya selalu update dengan perkembangan zaman.

4. Olahraga

Selain kegiatan di depan laptop, saya juga suka olahraga ringan seperti jogging dan bermain futsal. Olahraga membantu menjaga kesehatan tubuh sekaligus membuat pikiran lebih segar.

5. Desain Grafis

Saya tertarik dengan dunia desain grafis. Kadang saya mencoba membuat desain sederhana menggunakan aplikasi seperti CorelDRAW atau Canva untuk mengasah kreativitas.

Menurut saya, memiliki hobi yang beragam sangat penting untuk menyeimbangkan aktivitas akademik dengan kegiatan pribadi. Hobi membuat saya lebih semangat belajar dan terus mengembangkan potensi diri.
